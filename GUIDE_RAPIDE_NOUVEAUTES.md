## 📚 1. Plateforme d'Apprentissage (LMS)

**Accès**: Hugin → Plateforme d'Apprentissage

### Nouveautés
- ✅ Cours structurés en sections
- ✅ Leçons avec types variés (vidéo, document, quiz)
- ✅ Système de devoirs complet
- ✅ Progression détaillée
- ✅ Tableau de notes automatique
- ✅ Calendrier académique

### Comment l'utiliser ?
1. Cliquez sur un cours pour l'ouvrir
2. Explorez les sections (cliquez pour déplier)
3. Complétez les leçons
4. Soumettez vos devoirs
5. Consultez vos notes

---

## 📝 2. QCM & Examens en Ligne

**Accès**: Hugin → QCM Multi-Disciplines

### Nouveautés
- ✅ 5 types de questions (MCQ, choix multiples, vrai/faux, réponse courte, essai)
- ✅ Système anti-triche:
  - Détection changement d'onglet ⚠️
  - Mode plein écran
  - Webcam (optionnel)
- ✅ Timer avec alertes
- ✅ Correction automatique
- ✅ Explications détaillées

### Comment l'utiliser ?
1. Choisissez une discipline
2. Sélectionnez le mode (Révision ou Examen)
3. Cliquez sur "Commencer"
4. Répondez aux questions
5. **Important**: Ne changez pas d'onglet pendant l'examen !
6. Soumettez et consultez vos résultats

### ⚠️ Attention
- En mode examen, chaque changement d'onglet est détecté
- Vous verrez une alerte rouge si vous changez d'onglet
- Le nombre de violations est enregistré

## ☁️ 3. Stockage Cloud Professionnel

**Accès**: Hugin → Stockage Cloud

### Nouveautés
- ✅ Versioning complet (historique des modifications)
- ✅ Partage avec permissions (lecture, édition, admin)
- ✅ Workflow d'approbation (brouillon → révision → approuvé)
- ✅ Signatures électroniques
- ✅ Tags et favoris
- ✅ Chiffrement des fichiers sensibles
- ✅ Deux modes d'affichage (grille/liste)

### Comment l'utiliser ?
1. **Uploader un fichier**: Cliquez sur "Téléverser"
2. **Voir les versions**: Cliquez sur l'icône historique
3. **Partager**: Cliquez sur l'icône partage
4. **Organiser**: Ajoutez des tags
5. **Favoris**: Cliquez sur l'étoile
6. **Changer de vue**: Grille ou Liste

### Icônes à connaître
- 🔒 = Fichier chiffré
- ⭐ = Favori
- 🔄 = Plusieurs versions
- ✓ = Signé électroniquement
- 🔗 = Partagé

## 🤖 4. Mímir - Assistant Scientifique Global

### 🎯 La Grande Nouveauté !

**Mímir est maintenant accessible depuis TOUTES les pages !**

### Comment ça marche ?
1. **Regardez en bas à droite** de votre écran
2. Vous verrez un **bouton bleu qui pulse** avec une icône de robot
3. **Cliquez dessus** pour ouvrir le chat
4. **Posez votre question** scientifique
5. **Appuyez sur Entrée** pour envoyer
6. Mímir vous répond instantanément !

### Fonctionnalités
- ✅ Accessible depuis n'importe quelle page
- ✅ Chat interactif avec historique
- ✅ Réponses scientifiques instantanées
- ✅ Mode réduit (cliquez sur `-`)
- ✅ Envoi rapide (Entrée)
- ✅ Nouvelle ligne (Shift+Entrée)

### Exemples de questions
- "Comment faire une PCR ?"
- "Quelle température pour l'hybridation ?"
- "Explique-moi le Western Blot"
- "Comment calculer une dilution 1:10 ?"
- "Quels tampons pour électrophorèse ?"

### Astuces
- 💡 Le bouton pulse pour attirer votre attention
- 💡 Cliquez sur `-` pour réduire la fenêtre
- 💡 Cliquez sur `□` pour agrandir
- 💡 Cliquez sur `×` pour fermer
- 💡 L'historique est conservé pendant votre session

## ❓ FAQ

### Q: Mímir ne répond pas ?
**R**: Vérifiez que vous avez configuré votre clé API dans le module Mímir (`/hugin/ai-assistant`)

### Q: Je ne vois pas le bouton Mímir ?
**R**: Rechargez la page (Ctrl+R). Le bouton est en bas à droite.

### Q: L'examen détecte mes changements d'onglet ?
**R**: C'est normal ! C'est le système anti-triche. Restez sur l'onglet de l'examen.

### Q: Comment restaurer une ancienne version d'un fichier ?
**R**: Dans Cloud Storage, cliquez sur l'icône historique, puis "Restaurer" sur la version souhaitée.

### Q: Puis-je utiliser Mímir hors ligne ?
**R**: Mímir utilise une base de connaissances locale, donc oui pour les questions basiques. Pour les questions avancées, une connexion internet est nécessaire si vous utilisez une API externe.
