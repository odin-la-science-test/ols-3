2. Accédez à **Hugin Scholar** depuis le menu

## 📚 Module 1: Phénotypes de Résistance

### Comment l'utiliser:
1. Cliquez sur **"Phénotypes de Résistance"** dans Hugin
2. Utilisez la barre de recherche pour trouver un mécanisme
3. Filtrez par catégorie: Enzymatique, Cible, Perméabilité, Efflux
4. Cliquez sur une carte pour voir les détails complets

### Exemple d'utilisation:
```
Recherche: "BLSE"
→ Voir les gènes: blaCTX-M, blaSHV, blaTEM
→ Antibiotiques: Pénicillines, Céphalosporines 3G
→ Détection: Test de synergie, PCR
```

### Cas d'usage:
- Révision avant examen de microbiologie
- Préparation de TP sur antibiogramme
- Compréhension des mécanismes de résistance
- Aide à l'interprétation des résultats

---

## 🔬 Module 2: Fiches Machines

### Comment l'utiliser:
1. Cliquez sur **"Fiches Machines"** dans Hugin
2. Parcourez les catégories d'équipements
3. Cliquez sur un équipement pour voir:
   - Spécifications techniques
   - Applications
   - Consignes de sécurité
   - Instructions de maintenance

### Exemple d'utilisation:
```
Équipement: Microscope Optique
→ Grossissement: 40x - 1000x
→ Applications: Observation cellulaire, Histologie
→ Sécurité: Porter des gants, ne pas toucher les lentilles
```

### Cas d'usage:
- Préparation avant utilisation d'un équipement
- Révision des protocoles de sécurité
- Dépannage en cas de problème
- Formation aux nouveaux équipements

---

## 🧠 Module 3: QCM Multi-Disciplines

### Comment l'utiliser:
1. Cliquez sur **"QCM Multi-Disciplines"** dans Hugin
2. Choisissez une discipline (12 disponibles)
3. Sélectionnez un mode:
   - **Mode Révision:** Corrections immédiates + explications
   - **Mode Examen:** Chronométré, sans corrections

### Exemple de session:
```
Discipline: Biologie cellulaire
Mode: Révision
Question: "Quelle est la fonction principale des mitochondries ?"
→ Réponse: Production d'ATP
→ Explication: Les mitochondries sont les centrales énergétiques...
```

### Cas d'usage:
- Révision quotidienne (10-15 min)
- Préparation aux examens
- Auto-évaluation des connaissances
- Identification des points faibles

### Conseils:
- Commencez par le mode Révision
- Faites 1 discipline par jour
- Notez vos scores pour suivre votre progression
- Relisez les explications des mauvaises réponses

## 🎓 Module 4: Plateforme d'Apprentissage (LMS)

### Comment l'utiliser:
1. Cliquez sur **"Plateforme d'Apprentissage"** dans Hugin
2. Naviguez entre les onglets:
   - **Mes Cours:** Voir vos cours et progression
   - **Devoirs:** Gérer vos travaux
   - **Notes:** Consulter vos résultats

### Exemple de cours:
```
Cours: Biologie Moléculaire Avancée
Progression: 65% (13/20 leçons)
Note: 13/20
Instructeur: Dr. Martin
```

### Fonctionnalités:
- Suivi de progression visuel
- Barre de progression par cours
- Calcul automatique des notes
- Organisation par catégorie

### Cas d'usage:
- Suivre plusieurs cours simultanément
- Planifier ses révisions
- Consulter ses notes
- Identifier les cours à rattraper

---

## ☁️ Module 5: Stockage Cloud

### Comment l'utiliser:
1. Cliquez sur **"Stockage Cloud"** dans Hugin
2. Utilisez les boutons:
   - **Téléverser:** Ajouter des fichiers
   - **Vue Grille/Liste:** Changer l'affichage
3. Actions sur les fichiers:
   - Télécharger
   - Partager
   - Supprimer

### Organisation:
```
📁 Protocoles TP
📁 Images_Microscope
📄 Rapport_Microbiologie.pdf (2.5 MB)
📄 Données_Expérience.xlsx (1.2 MB)
```

### Cas d'usage:
- Stocker vos rapports de TP
- Partager des documents avec votre groupe
- Sauvegarder vos données d'expérience
- Accéder à vos fichiers depuis n'importe où

### Conseils:
- Organisez vos fichiers par dossiers
- Utilisez des noms de fichiers clairs
- Partagez uniquement avec les personnes concernées
- Vérifiez régulièrement votre espace disponible

## 🎯 Workflows Recommandés

### Workflow 1: Préparation d'un TP
1. **Fiches Machines** → Consulter l'équipement à utiliser
2. **Phénotypes de Résistance** → Réviser la théorie
3. **Cloud Storage** → Télécharger le protocole
4. **Cahier de Labo** → Prendre des notes pendant le TP

### Workflow 2: Révision d'Examen
1. **QCM Multi-Disciplines** → Tester ses connaissances
2. **Phénotypes de Résistance** → Approfondir les points faibles
3. **LMS** → Consulter les cours et notes
4. **Cloud Storage** → Accéder aux fiches de révision

### Workflow 3: Rédaction de Rapport
1. **Cahier de Labo** → Récupérer les notes d'expérience
2. **Traitement de Texte** → Rédiger le rapport
3. **Cloud Storage** → Sauvegarder et partager
4. **LMS** → Soumettre le devoir
